Требования к продукту

**Название продукта:** ReadMe

**Сроки проекта:** 9.03.2026-10.06.2026

**Суть продукта (в 1-2 предложениях):** Система хранения и просмотра учебных материалов, связанная с системами проверки знаний, которая удобна для заказчика.

**Полное описание продукта:** 

Веб-приложение для публикации учебных материалов по различным курсам\* факультета ИВТ.

Приложение будет основано на клиент-серверной архитектуре. Хранилищем для информации о пользователях и загружаемых материалов выступит база данных.

Основные отличия от Moodle: ориентированность на удобное предоставление материала (возможность добавления файлов в формате markdown, поддержка TeX и редактирования текста непосредственно в приложении), разработка с учётом обратной связи от преподавателей, интеграция с уникальными сервисами проверки знаний, разработанными студентами факультета (прим.: система CheckMe для тестирования sql-скриптов). 

\* приложение пишется в первую очередь для курса “Базы данных” с возможностью добавления других курсов при будущей поддержке приложения

**Описание (предполагаемой) целевой аудитории продукта:** преподаватели и студенты различных образовательных учреждений.

**Требуемые ресурсы:** 

1. Компьютеры и ноутбуки для разработки и написания кода.   
2. Среды разработки VSCode, PyCharm, IDEA.   
3. Приложение для разработки дизайна: Figma, Paint, Photoshop.   
4. Сервер с PostgreSQL 18\.  
5. Приложения для виртуальных контейнеров: Docker Desktop.  
6. Сервер для развертывания итогового проекта.
